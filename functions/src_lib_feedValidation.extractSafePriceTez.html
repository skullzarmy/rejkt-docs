    Function extractSafePriceTez

    • extractSafePriceTez(item: any): null | string

      Safely extracts and formats price in tez from any feed item type

      This function handles the complexity of extracting price information from different feed item types, dealing with various data structures and formats. It includes validation to ensure only valid prices are extracted and formatted consistently.

      The function handles these scenarios:

      1. Items with listings_active array (finds lowest valid price)
      2. Items with direct 'price' property (converts from mutez)
      3. Items with direct 'price_xtz' property (uses as-is)

      Parameters

      • item: any

        Feed item (BaseListing, ArtistListing, or GalleryToken)

      Returns null | string

      Formatted price string (e.g. "10.50") or null if price not available/valid

      Example

      const price = extractSafePriceTez(listing);
      const displayPrice = price ? `${price} tez` : 'Not for sale';
