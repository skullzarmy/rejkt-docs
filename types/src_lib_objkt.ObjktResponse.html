    Type Alias ObjktResponse<T>

    ObjktResponse:
        | { data: T; success: true }
        | { attempts: number; error: string; success: false }

    Response type for OBJKT API requests with success/failure state

    This discriminated union type represents the two possible outcomes of an API request:

    1. Success: Contains the requested data
    2. Failure: Contains error information and retry attempts count

    Using this type allows for type-safe handling of both success and error cases with proper TypeScript narrowing.

    Type Parameters

    • T

      The expected data type on successful responses

    Example

    const response = await fetchObjktGraphQL<ListingsData>(query);
    if (response.success) {
      // TypeScript knows response.data exists and is of type ListingsData
      processListings(response.data.listings);
    } else {
      // TypeScript knows response.error exists
      console.error(`Request failed: ${response.error}`);
    }
