    Interface StorageSizeBreakdown

    Detailed size breakdown of a storage item

    This interface provides a comprehensive breakdown of storage usage for a single key-value pair stored in IndexedDB, including overhead calculations and detailed size analysis of complex object properties.

    interface StorageSizeBreakdown {
        keySize: number;
        metadataSize: number;
        totalSize: number;
        valueDetails?: { [key: string]: number };
        valueSize: number;
    }
