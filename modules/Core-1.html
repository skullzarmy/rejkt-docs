    Module Core

    ThemeToggle Component

    A button component that allows users to toggle between light and dark themes. Features a sun/moon icon that changes based on the current theme, and includes a tooltip for better UX.

    Features:

    • Automatic theme detection
    • Smooth icon transitions
    • Accessible button controls
    • Tooltip guidance
    • SSR hydration handling
    • Responsive design

    The component handles SSR by mounting only on the client side to prevent hydration mismatches, and uses next-themes for theme management.

    Example

    // Basic usage in header
    <header>
      <ThemeToggle />
    </header>

    // With custom container
    <div className="theme-controls">
      <ThemeToggle />
      <OtherControls />
    </div>
