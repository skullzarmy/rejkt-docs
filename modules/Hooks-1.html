    Draft Filters Hook

    A React hook that manages draft state for filters, allowing users to make temporary changes before committing them. Supports multiple filter types including toggles, ranges, and tags.

    Features:

    • Local draft state management
    • Filter type support (toggle, range, tag)
    • Change detection
    • Synchronized updates with context
    • Type-safe operations
    • Timestamp tracking

    Example

    // Basic usage
    const {
      draftFilters,
      toggleFilter,
      setRange,
      setTags,
      unsavedChanges
    } = useDraftFilters(initialFilters)

    // Toggle a filter
    toggleFilter("price")

    // Set a range
    setRange("price", [0, 100])

    // Set tags
    setTags("category", ["art", "photography"])

    // Check for unsaved changes
    if (unsavedChanges) {
      // Show save/cancel buttons
    }
